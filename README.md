Warung Mama (Website Sederhana untuk Beli Sembako)

| File | Fungsinya |
| :--- | :--- |
| `index.php` | Halaman utama toko (tempat daftar sembako ditampilkan). |
| `beli.php` | Menangani logika ketika pelanggan menekan tombol 'Beli'. |
| `process.php` | Menangani pemrosesan data, misalnya menyimpan transaksi ke database. |
| `config.php` | Tempat mengatur kredensial database dan informasi toko lainnya. |
| `style.css` | Semua gaya tampilan utama aplikasi. |
